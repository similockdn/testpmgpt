BẢN CẬP NHẬT HOA HỒNG - V58

Nội dung cập nhật:
1. Hoa hồng Sale tính trên: Tiền hàng + Phụ thu - Chiết khấu.
2. Xuất Excel hoa hồng theo đúng 01 nhân viên đang chọn, không xuất tất cả nhân viên một lần.
3. Bổ sung cột Model và Số lượng trong:
   - Chi tiết hoa hồng trên giao diện.
   - File Excel hoa hồng theo nhân viên.
   - Excel tổng hoa hồng.
4. Nếu 1 phiếu có nhiều model, hệ thống gom model và hiển thị chi tiết số lượng theo từng model.

Đã kiểm tra:
- app.js không lỗi cú pháp bằng node --check.
- index.html đã cập nhật header bảng hoa hồng.
- Không thay đổi dữ liệu Firebase/database.

# V80 - Thiết kế lại Dashboard

- Làm mới dashboard theo giao diện ERP hiện đại.
- KPI hiển thị dạng card có icon, bo góc, shadow nhẹ.
- Bộ lọc Tháng này / 7 ngày / Năm nay được đưa vào hero header.
- Khu biểu đồ được bố trí lại dạng dashboard rõ ràng.
- Các bảng Đơn hàng, Công nợ, Hàng sắp hết, Hiệu suất nhân viên được làm lại giao diện.
- Bổ sung CSS override mạnh để tránh lỗi giao diện rơi về HTML mặc định.

# CHANGELOG V52 - UX FIX

## Đã hiệu chỉnh

### 1. Công nợ
- Rút gọn bảng công nợ.
- Gom mã KH, SĐT, địa chỉ vào ô khách hàng.
- Gom sản phẩm vào ô riêng, giới hạn độ rộng để bảng dễ nhìn.
- Bộ lọc Đang nợ / Đã tất toán / Tất cả sẽ ẩn/hiện đúng bảng cần xem.
- Thống kê công nợ hiển thị dạng 3 ô rõ hơn.

### 2. Bảng giá
- Thu nhỏ item sản phẩm trong khung chọn model.
- Hiển thị dạng chip: Model, tên sản phẩm, nhóm sản phẩm.
- Tăng khả năng hiển thị nhiều model trên một màn hình.
- Bảng nhập giá có khung cuộn riêng, tránh kéo trang quá dài.

### 3. Báo cáo
- Sửa các tab báo cáo không chạy.
- Tab Doanh thu hiển thị các bảng doanh thu, sản phẩm bán ra, hàng trả.
- Tab Lợi nhuận hiển thị phân tích lợi nhuận, chi phí, lương.
- Tab Tồn kho / Công nợ / Hoa hồng / Bảo hành chuyển đúng module.

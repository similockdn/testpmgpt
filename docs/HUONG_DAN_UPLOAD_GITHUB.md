# HƯỚNG DẪN UPLOAD GITHUB

## Cách an toàn

1. Giải nén ZIP.
2. Vào thư mục đã giải nén.
3. Chọn toàn bộ file và thư mục bên trong, không chọn chính thư mục cha.
4. Kéo thả vào GitHub Upload.
5. Commit: `ERP Stable Final`.
6. Chờ GitHub Pages cập nhật 1-3 phút.
7. Mở trình duyệt và nhấn Ctrl+F5 để xóa cache.

## Không được làm

- Không đổi `firebase-config.js` sang project khác.
- Không bấm Clear Data.
- Không sửa Firestore Rules nếu chưa test.

## Nên làm trước khi lên bản chính

- Upload vào repo test hoặc branch test.
- Test checklist trong `QA_REPORT_FINAL.md`.
- Khi ổn mới upload vào bản chính.

# V53 - Price UI Redesign

- Redesign module Bảng giá theo dạng phiếu nhập liệu giống phiếu bán hàng.
- Tách 2 tab rõ ràng: Bảng giá bán hàng / Bảng giá vốn.
- Form thông tin bảng giá đặt phía trên, danh sách bảng giá và bảng sản phẩm đặt phía dưới.
- Item sản phẩm thu nhỏ dạng chip, tối ưu cho nhiều model SIMILOCK.
- Không thay đổi schema Firestore để giữ tương thích dữ liệu cũ.
